File Handling
-------------
Data = ascii characters = no need of drivers


open a file in 
-----------

1. read mode (r) - you can only read the contents of the file, if the file is not present then throws an exception FileNotFound

2. write mode (w) - if the file is existing then delete the contents of the file and write newly into the file. if the is not existing then create a new file and write into it

3. append mode (a) - if the file exists then append data into the file without disturbing the previous content. if the is not existing then create a new file and write into it
